## How to add your resume

1. Export or save your resume as a PDF
2. Rename it to `Burra_Venkatesh_Resume.pdf`
3. Place it in this folder: `assets/resume/`

## Tips for a strong fresher resume

- Keep it to 1 page
- Highlight your Django college project with tech stack and features
- List skills matching the portfolio (Python, Django, HTML, CSS, JS)
- Include education with CGPA
- Add GitHub link: https://github.com/venkatesh-burra
- Include certifications listed in the portfolio
